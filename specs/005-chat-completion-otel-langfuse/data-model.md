# Data Model: Chat Completion with Observability

**Feature**: `005-chat-completion-otel-langfuse` | **Date**: 2026-05-28

## Entities

### ChatRequest

Input to `POST /v1/chat/completions`.

| Field | Type | Required | Constraints | Notes |
|---|---|---|---|---|
| `model` | string | yes | Must match a name in model_list | LiteLLM returns 400 if unknown |
| `messages` | array[Message] | yes | minimum 1 element | LiteLLM returns 400 if empty |
| `stream` | boolean | no | default: `false` | `true` is out of scope for this feature |
| `metadata` | object[Metadata] | no | ‚Äî | All sub-fields optional |

#### Message

| Field | Type | Required | Constraints |
|---|---|---|---|
| `role` | string | yes | `"user"` \| `"assistant"` \| `"system"` |
| `content` | string | yes | non-empty |

#### Metadata

| Field | Type | Required | Notes |
|---|---|---|---|
| `team` | string | no | Tags Phoenix span and Langfuse trace |
| `request_id` | string | no | Tags Phoenix span and Langfuse trace |
| `prompt_name` | string | no | Links Langfuse trace to named `production` prompt version |

---

### ChatResponse

HTTP 200 response body. Returned as a complete JSON document (non-streaming).

| Field | Type | Constraints | Notes |
|---|---|---|---|
| `id` | string | non-empty | LiteLLM-generated completion ID |
| `object` | string | `"chat.completion"` | OpenAI-compatible envelope |
| `model` | string | echoes the requested model name | ‚Äî |
| `choices` | array[Choice] | exactly 1 element (non-streaming) | ‚Äî |
| `usage` | Usage | ‚Äî | Token counts |

#### Choice

| Field | Type | Constraints |
|---|---|---|
| `index` | integer | `0` |
| `message` | Message | role: `"assistant"` |
| `finish_reason` | string | `"stop"` \| `"length"` \| `"content_filter"` |

#### Usage

| Field | Type | Constraints |
|---|---|---|
| `prompt_tokens` | integer | > 0 |
| `completion_tokens` | integer | > 0 |
| `total_tokens` | integer | `prompt_tokens + completion_tokens` |

**Response headers** (added by Kong):
- `X-Request-ID`: correlation ID (UUID, generated by Kong correlation-id plugin)
- `X-Platform: inference-platform`
- `X-API-Version: 1`

### ObservabilitySpan

A record emitted to Phoenix Arize via the `arize_phoenix` LiteLLM callback. Not a request/response field ‚Äî internal platform event.

| Attribute | OpenInference Key | Source |
|---|---|---|
| Model name | `llm.model_name` | requested `model` field |
| Prompt tokens | `llm.token_count.prompt` | `usage.prompt_tokens` in provider response |
| Completion tokens | `llm.token_count.completion` | `usage.completion_tokens` in provider response |
| Team tag | `metadata.team` | `request.metadata.team` |
| Request ID tag | `metadata.request_id` | `request.metadata.request_id` or X-Request-ID |

**Span kind**: `LLM` (OpenInference convention)

---

### LangfuseTrace

A record created in Langfuse via the `langfuse` LiteLLM callback. Not a request/response field ‚Äî internal platform event.

| Field | Source | Notes |
|---|---|---|
| `traceId` | correlation ID | Matches `X-Request-ID` header |
| `model` | requested `model` | ‚Äî |
| `promptTokens` | `usage.prompt_tokens` | ‚Äî |
| `completionTokens` | `usage.completion_tokens` | ‚Äî |
| `metadata.team` | `request.metadata.team` | Optional |
| `metadata.request_id` | `request.metadata.request_id` | Optional |
| `promptName` | `request.metadata.prompt_name` | Optional; triggers prompt version link |
| `promptVersion` | resolved from Langfuse | `production` label of named prompt |

---

### PromptVersion

Stored in Langfuse (pre-existing). Referenced at request time by `prompt_name`.

| Field | Type | Notes |
|---|---|---|
| `name` | string | Matches `metadata.prompt_name` in request |
| `label` | string | Must be `"production"` to be linked |
| `version` | integer | Monotonically increasing |
| `content` | string | System prompt text (never logged or persisted outside Langfuse) |

**Lifecycle**: Managed by `scripts/prompt-register.py` ‚Üí evaluate ‚Üí `scripts/prompt-promote.py`.
